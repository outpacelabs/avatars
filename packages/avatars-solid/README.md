# @outpacelabs/avatars-solid

SolidJS component for [@outpacelabs/avatars](https://www.npmjs.com/package/@outpacelabs/avatars).

```sh
pnpm add @outpacelabs/avatars @outpacelabs/avatars-solid
```

```tsx
import { GradientAvatar } from "@outpacelabs/avatars-solid";

<GradientAvatar id={7} size={64} />
<GradientAvatar seed="user-123" size={48} preview />
```

See [`@outpacelabs/avatars`](https://www.npmjs.com/package/@outpacelabs/avatars) for how to serve the asset files.
